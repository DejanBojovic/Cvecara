Što se tiče naše web apliakacije, imaćemo web server odnosno sbb online host server kako bi naša web prodavnica radila I broj poseta bio na većem nivou. Preko tog servera za našu web prodavnicu koristitimo laravel,html,css, react.js kao I mysql sto se tiče baze podataka. Laravel koristimo jer je najpoznatiji php-ov framework i ujedno I najbolji, ima razne ugradjene funkcije koje pomažu da se sama aplikacija izradi I bude fenomenalna,Laravel framework jednostavno rečeno je struktura koju možete koristiti da nešto napravite. Omogućava vam upotrebu različitih komponenti, komunikaciju sa spoljnim API-jevima i definisanje strukture vaše aplikacije, smanjuje slozenost aplikacije, kod je dosta čist I olakšava testiranje same aplikacije, da ne bi sada zalazili u dubinu, lavarel je u suštini vrhunski alat za izradu web aplikacija.Za bazu podataka koristimo MySql kako bi podatke povezali za web prodavnicu I naši proizvodi bili prikazani, u njoj se nalaze proizvodi, količina, cena, opis proizvoda, podaci o mušterijama i administrativnom delu osoblja kao I ostale stvari koje su potrebne za našu web prodavnicu kako bi funkcionisala na vrhunskom nivou. Što se tiče izgleda aplikacije, koristimo html I css kao I  javascript odnosno njegov framework react.js, html I css je markup jezik koji koristimo kako bi dizajnirali I napravili stranicu kako želimo da izgledaju odnosno pomoću njih naša web prodavnica izgleda kako smo zamislili I kako želimo da izgleda, to je deo koji naši korisnici vide i zbog kojih se zadržavaju I gledaju naše proizvode I stranicu. U suštini naš sistem za web prodavnicu odnosno web aplikaciju koriste html,css, react.js, laravel i mysql bazu podataka kako bi naša aplikacija izgledala vrhunski I funkcionalno.Što se tiče operativnih sistema, taj deo ne utiče na nas jer ako je na nekom sistemu podržan google, safari, firefox ili neki drugi browser moći će pristupiti nasoj web prodavnici. Sistem omogućava odredjene funkcionalnosti mušterijama, kao i administrativnom delu. Mušterije se mogu registrovati, prijaviti i poručiti odredjene proizvode i sastaviti narudžbinu.Svaki uredjaj koji poseduje browser moze ući na našu web podavnicu i kupovati proizvode. Administrativni deo, kojem prodavci imaju pristup, se koristi za validaciju narudžbina I praćenja stanja resursa cvećare.Sto se tiče prodavaca, aplikacija je prijateljski orjentisana, odnosno moći će je koriste sve osobe jer je napravljena da olakša posao ali I odradi sve potrebne stvari.Naš sistem ima guest korisnika koji nema nalog i gleda naš sajt, imamo mušteriju koja je napravila nalog i može pored gledanja i kupovati i imamo administratora odnosno prodavca koji ima pristup sajtu i bazi podataka kako bi ažurirao sve sto je potrebno.
